# Code and Data Availability wording

## Manuscript-ready code availability statement
The code supporting the GeoHybridGNN workflow will be made publicly available in a GitHub repository upon publication, with an archived release on Zenodo.

## Manuscript-ready data availability statement
The processed code used to generate the published results will be made publicly available. Some input geoscientific datasets are derived from external or third-party sources and should be obtained from the original providers cited in the manuscript. Additional derived data may be available from the corresponding author upon reasonable request.

## If you want full openness now
The code supporting the GeoHybridGNN workflow is publicly available in the project GitHub repository, and the archived release is available through Zenodo. Data availability varies by source layer; redistributed data are limited to materials that can be shared under their source licenses.
