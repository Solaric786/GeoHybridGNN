# Code and Data Availability

## Manuscript-ready statement

The public code snapshot associated with the GeoHybridGNN project is available in the project GitHub repository. An archived release can be created through Zenodo for long-term citation and versioning.

Some geoscientific input datasets used in the study are derived from external or third-party sources and should be obtained from the original providers cited in the manuscript. This public repository does not redistribute restricted or non-shareable source data. Additional derived materials may be available from the corresponding author upon reasonable request and subject to source-data sharing constraints.

## Notes

- Publicly shared materials in this repository are limited to files prepared for release.
- Large intermediate outputs, restricted datasets, and private local project assets are not included.
- Zenodo archival should be linked to a tagged GitHub release when the first formal public version is finalized.
